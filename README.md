# `go-ios` Web App

Web application for [go-ios](https://github.com/danielpaulus/go-ios).

---

Based on the following stack:

1. [Nuxt 3](https://v3.nuxtjs.org/)
2. [TailwindCSS](https://tailwindcss.com)
3. [PrimeVue](https://www.primefaces.org/)
